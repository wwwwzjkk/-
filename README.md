# 广州好六网络科技有限公司网站

## 项目说明
这是为广州好六网络科技有限公司设计的企业网站，包含完整的五个页面。

## 文件结构
```
好六网络公司网站/
├── index.html       # 首页
├── about.html       # 关于我们
├── services.html    # 产品服务
├── cases.html       # 作品案例
├── contact.html     # 联系我们
├── style.css        # 样式文件
├── script.js        # JavaScript文件
└── README.md        # 项目说明
```

## 页面介绍

### 1. 首页 (index.html)
- 品牌横幅展示
- 公司简介
- 核心优势展示
- 服务预览

### 2. 关于我们 (about.html)
- 公司介绍
- 企业文化（愿景、使命、价值观）
- 发展历程时间线
- 核心优势详情

### 3. 产品服务 (services.html)
- 服务介绍总览
- 六大服务详细说明：
  * 信息流广告投放
  * 短视频营销
  * 社交媒体运营
  * 品牌策划
  * 数据分析
  * 创意设计
- 服务流程展示

### 4. 作品案例 (cases.html)
- 精选案例展示（6个案例卡片）
- 服务成果数据展示
- 客户评价

### 5. 联系我们 (contact.html)
- 联系方式信息
- 在线留言表单
- 合作流程说明
- 常见问题FAQ
- 地图位置

## 特色功能
- 响应式设计，适配各种设备
- 现代化渐变色设计风格
- 平滑的页面交互效果
- FAQ折叠展开功能
- 表单验证功能

## 使用说明
1. 直接用浏览器打开 `index.html` 即可浏览网站
2. 所有页面都已完成并互相链接
3. 可根据实际需要替换图片占位符
4. 可修改联系方式等具体信息

## 技术栈
- HTML5
- CSS3 (使用Flexbox和Grid布局)
- 原生JavaScript

## 自定义建议
1. 将图片占位符替换为实际的公司图片
2. 更新具体的联系方式（电话、地址、邮箱）
3. 添加公司Logo图片
4. 根据品牌VI调整配色方案
5. 接入实际的地图API（如百度地图、高德地图）

## 浏览器兼容性
- Chrome (推荐)
- Firefox
- Safari
- Edge
- 移动端浏览器

## 注意事项
- 表单提交目前为前端验证，需要后端API支持才能真正提交数据
- 图片使用emoji作为临时占位，建议替换为实际图片
- 建议将CSS和JS文件进行压缩以提升加载速度
